# syncotrope

## Prerequisites
- Install [NVM](https://github.com/nvm-sh/nvm)
- Install node version 20 with `nvm install 20`

## Setup
```
git clone git@github.com:nickhobbs94/syncotrope.git
cd syncotrope
nvm use
npm install
```

## Build
```
npm run build
```

## Run
```
npm run start
```

## Test
```
npm run test
```
